# 哔哩哔哩 BWS（BW乐园）活动预约工具
> 预约前请确保您已拥有可用的 BW2026 门票，否则无法进行活动预约

### 版本说明
- 支持扫码登录
- 支持 Cookie 登录
- 支持可视化项目选择
- 支持二次付费项目显示（如签售）

### 特别说明
**如果你的设备已经提前完成校时，你可以选择使用本机时间而非程序提供的 NTP 时间，因为服务器等设施的时间可能更准确。**

**当你使用本机时间，但与 NTP 服务器的时间差 > 0.7 秒时，会强制切换到 NTP 时间，如果不想强制切走开抢前需要自己给设备校时。**
