# BWS Ticket
> 支持在 BW 乐园中预约 BiliBili World (BW) 线下活动及商品购买资格（签售、见面会、玩家活动、限量购买资格等）

**预约前请确保您已拥有可用的 BW2026 门票，否则无法进行活动预约**

**在黄牛泛滥的现状下，本项目将致力于拉高普通观众的抢票机会，本项目不会、也不参与任何收费营利活动**

## 版本说明
- 支持扫码、Cookie 登录
- 可视化的项目选择功能
- 支持 BW 乐园活动预约
- 支持 BW 限量商品抢购
- 支持查看已预约的项目及详情
- 支持二次付费项目显示（如：签售）
